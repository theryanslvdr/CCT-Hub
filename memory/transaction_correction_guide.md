# Transaction Correction Guide

This guide explains how members and admins can correct transaction amounts in CrossCurrent Hub.

---

## For Members: Self-Edit Your Transactions

### When Can You Edit?
- You can edit your **last 2 deposits or withdrawals** only
- You have a **48-hour window** from when the transaction was created
- If an admin has already corrected a transaction, you cannot edit it further

### Step-by-Step

1. **Go to the Profit Planner page**
   - Navigate to "Profit Planner" from the sidebar

2. **Find "My Recent Transactions"**
   - Scroll down to the "My Recent Transactions" section on the page
   - You'll see your recent deposits and withdrawals listed

3. **Click "Edit" on the transaction you want to correct**
   - Each editable transaction has a blue "Edit" button on the right
   - If you see a lock icon instead, the 48-hour window has expired

4. **Enter the correct amount**
   - A dialog will appear showing the current amount
   - Type the correct dollar amount (just the number, no $ sign needed)
   - Optionally, type a reason for the change (e.g., "Entered wrong amount")

5. **Click "Update"**
   - Your balance will update immediately
   - The transaction will show an "Edited" badge
   - The admin will be notified of the change

### Rules
- Only deposits and withdrawals can be edited (not trades or commissions)
- You can only edit once — make sure the amount is correct
- The 48-hour countdown starts from when the transaction was originally created

---

## For Admins: Correct Any Member's Transaction

### When Should You Correct?
- When a member reports a wrong amount
- When you spot an incorrect deposit or withdrawal during reconciliation
- When a member's 48-hour edit window has expired and they need a fix

### Step-by-Step

1. **Go to Admin > Transactions**
   - Open the Admin section in the sidebar
   - Click "Transactions" to view all member transactions

2. **Find the transaction**
   - Use the search bar to find a specific member
   - Filter by type (Deposit/Withdrawal) or date range
   - Scroll through the list to locate the transaction

3. **Click the pencil icon**
   - On the right side of each transaction row, click the pencil (edit) icon
   - This opens the correction dialog

4. **Enter the corrected amount**
   - The dialog shows the member's name and the current amount
   - Type the correct dollar amount
   - Add a reason for the correction (e.g., "Member reported wrong deposit amount")

5. **Click "Correct"**
   - The member's balance updates immediately
   - An audit trail is recorded with:
     - Who made the correction (your name)
     - When it was corrected
     - The original amount vs. new amount
     - The reason provided
   - The transaction will show a "Corrected" badge
   - The member will be notified

### Admin Correction Rules
- Admin corrections **lock** the transaction — the member cannot edit it further
- All corrections are logged in the audit trail for accountability
- There is no time limit for admin corrections (unlike the 48-hour member limit)

| Feature | Member | Admin |

| **What can be edited** | Last 2 deposits/withdrawals | Any deposit/withdrawal |
| **Time limit** | 48 hours | No limit |
