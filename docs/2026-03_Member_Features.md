## Getting Started

### Creating Your Account
1. Visit the registration page
2. Enter your details:
   - Full Name
   - Email Address
   - Password (must be strong)
3. Verify your email if required
4. Log in to access your dashboard

### First Login
- Complete your profile information
- Set up your trading preferences
- Explore the guided tour (if available)
- Start tracking your first trade!

### Navigation
- **Sidebar** (desktop): Access all features
- **Bottom Menu** (mobile): Quick access to main features
- **Profile Dropdown**: Settings, profile, logout

## Dashboard

Your central hub for viewing financial overview.

### Key Cards

#### Account Value Card
- Shows your current account balance
- **Eye icon**: Toggle to hide/show value (privacy)
- Currency selector: USD, PHP, EUR, GBP

#### Total Profit Card
- Cumulative profit from all trades
- Green = profit, Red = loss

#### Performance Message
Contextual messages based on your trading:
- "You're on a 5-day winning streak!"
- "Great progress this month!"
- "Keep going, you're almost at your goal!"

### Rewards Preview
Quick view of your rewards status:
- Points balance
- Current level
- Monthly rank
- "Open Rewards & Store" button

### Quick Actions
- **Trade Now**: Jump to Trade Monitor
- **View History**: See recent trades
- **My Rewards**: Full rewards page

### For Licensees
If you have a license, you'll see:
- Year-by-Year Growth Projections (1, 2, 3, 5 years)
- Family Members table (if Honorary FA)
- Simplified view (no trading controls)

## Profit Tracker

Comprehensive view of your trading performance and projections.

### Summary Cards

| Card | What It Shows |
|------|---------------|
| Account Value | Current balance (with "Sync" to verify) |
| Deposits | Total deposited in selected currency |
| Total Profit | All-time trading profit |
| LOT Size | Your current lot size (Balance ÷ 980) |

**Privacy Feature:** Each card has an eye icon (👁) to individually hide that value.

### Projection Vision

#### Summary Tab
- Current account balance
- Calculated LOT size
- Daily profit projection
- Formula explanation

#### Monthly Table Tab
Day-by-day breakdown showing:

| Column | Description |
|--------|-------------|
| Date | Trading day |
| Balance Before | Account at day start |
| LOT Size | Lot for that balance |
| Target Profit | Expected profit (LOT × 15) |
| Actual Profit | What you actually earned |
| Commission | Commission received |
| Status | ✅ Trade done / 📅 Pending |

### Today's Signal
Shows the current trading signal:
- Product code (e.g., MOIL10)
- Direction (BUY/SELL)
- Time to trade
- "Trade Now" button

### Actions
- **Access Records**: View trade history
- **Reset Tracker**: Start fresh from a date
- **Export Data**: Download as CSV

#### Level

| Type | Cost | Protects |
|------|------|----------|

1. Select quantity (1-10)
2. Click "Buy"
3. Points deducted, freezes added to inventory
4. Auto-apply when you miss a day

## Community Forum

Get help and share knowledge with the community.

### Forum Overview
- Total posts
- Open questions
- Solved questions
- Top contributors

### Viewing Posts

Posts show:
- Title
- Author with role badge
- Status: Open (blue) or Solved (green)
- Tags
- View and comment counts
- Time posted

### Creating a Post

1. Click "New Post" button
2. Enter title (be descriptive)
3. Write detailed description
4. Add tags (comma-separated)
5. **Attach Images** (optional):
   - Up to 4 images
   - Max 2MB each
   - JPG, PNG, GIF, WebP supported
6. Check "Similar Posts" suggestions
7. Click "Post"

### Commenting

1. Open a post
2. Scroll to comment box
3. Write your reply
4. Attach images if helpful
5. Click "Reply"

### Voting

Help surface good answers:
- 👍 **Upvote**: Helpful comment
- 👎 **Downvote**: Not helpful
- See who voted (hover over count)
- Cannot vote on your own comments

### Best Answer

If you created the post:
1. Find the most helpful comment
2. Click "Mark as Best Answer"
3. That answerer earns 50 points!

### Closing Posts

When your question is answered:
1. Click "Close Post"
2. Select best answer
3. Choose active collaborators (15 pts each)
4. Confirm closure
